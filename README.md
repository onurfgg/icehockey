Welcome to the **Ice Hockey Analytics** repository. This project bridges the gap between raw sports data and strategic decision-making. By applying industrial engineering principles and advanced data science techniques, this repository aims to analyze game dynamics, evaluate player performance, and extract actionable insights from ice hockey datasets.

## 🎯 Project Objectives

* **Exploratory Data Analysis (EDA):** Deep dive into historical game data, identifying trends, correlations, and anomalies in team and player statistics.
* **Performance Metrics Evaluation:** Assessing key performance indicators (KPIs) that impact game outcomes beyond standard box score statistics.
* **Statistical Modeling & Visualization:** Creating intuitive, data-rich visualizations that communicate complex metrics to both technical and non-technical stakeholders (e.g., coaches, management).

## 📊 Key Features & Analysis

- **Data Preprocessing Pipelines:** Cleaning, merging, and structuring raw ice hockey datasets for robust analytical processing.
- **Advanced Visualizations:** Utilizing modern plotting libraries to map out shot heatmaps, momentum shifts, and player efficiency ratings.
- **Hypothesis Testing:** Statistically verifying assumptions about game strategies and their actual impact on winning probabilities.
